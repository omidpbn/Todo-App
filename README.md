## TaskBoard / Todo List Component Documentation

**Overview:**
The TaskBoard component is a full-featured Todo List system. It allows users to create, edit, delete, view, and move tasks between different columns. The board supports both light and dark modes and adapts to mobile screens with collapsible columns.

**Features:**

* **Create Task:** Users can add new tasks using a modal form.
* **Edit Task:** Existing tasks can be updated.
* **Delete Task:** Tasks can be removed individually.
* **View Tasks:** Tasks are displayed in columns (`backlog`, `todo`, `inprogress`, `done`).
* **Drag & Drop:** Tasks can be moved between columns using drag-and-drop.
* **Dark & Light Mode:** Automatically adapts to theme changes.
* **Responsive & Mobile Collapse:** On small screens, only the backlog column is shown by default; other columns can be expanded by tapping the header.
* **Scrollable Columns:** Each column scrolls independently when tasks overflow the available space.
* **Empty Column Placeholder:** Displays `"No tasks yet"` when a column has no tasks.

**Notes:**

* Column headers remain visible while scrolling tasks.
* Supports multiple tasks per column and ensures proper spacing and readability.

**نمای کلی:**
کامپوننت TaskBoard یک سیستم کامل برای لیست وظایف (Todo List) است. کاربران می‌توانند تسک‌ها را ایجاد، ویرایش، حذف، مشاهده و بین ستون‌ها جابجا کنند. این کامپوننت از حالت‌های دارک و لایت پشتیبانی می‌کند و در موبایل ستون‌ها به صورت جمع‌شونده نمایش داده می‌شوند.

**ویژگی‌ها:**

* **ایجاد تسک:** کاربران می‌توانند با استفاده از فرم مودال، تسک جدید اضافه کنند.
* **ویرایش تسک:** تسک‌های موجود قابل به‌روزرسانی هستند.
* **حذف تسک:** امکان حذف هر تسک به صورت جداگانه وجود دارد.
* **نمایش تسک‌ها:** تسک‌ها در ستون‌های مختلف (`backlog`, `todo`, `inprogress`, `done`) نمایش داده می‌شوند.
* **کشیدن و رها کردن (Drag & Drop):** کاربران می‌توانند تسک‌ها را بین ستون‌ها جابجا کنند.
* **حالت دارک و لایت:** کامپوننت به طور خودکار با تغییر تم هماهنگ می‌شود.
* **واکنش‌گرا و موبایل:** در صفحه‌نمایش کوچک، تنها ستون backlog به صورت پیش‌فرض باز است و بقیه ستون‌ها با لمس هدر قابل باز شدن هستند.
* **اسکرول ستون‌ها:** در صورت زیاد بودن تعداد تسک‌ها، ستون‌ها به صورت مستقل اسکرول می‌شوند.
* **نمایش ستون خالی:** زمانی که ستون تسکی ندارد، متن `"No tasks yet"` نمایش داده می‌شود.

**نکات:**

* هدر ستون‌ها در هنگام اسکرول تسک‌ها ثابت و قابل مشاهده باقی می‌مانند.
* پشتیبانی از چند تسک در هر ستون و حفظ فاصله مناسب برای خوانایی بهتر.
